# 🌟 AstroBit - Астрономическая платформа анализа криптовалют

**AstroBit** — это инновационная исследовательская веб-платформа, объединяющая анализ криптовалют с астрономическими и астрологическими данными для выявления потенциальных корреляций и паттернов между космическими событиями и рыночными движениями.

## 🌐 Живая версия

**[🚀 AstroBit Online](https://astrobit.online/)** — попробуйте платформу прямо сейчас!

## 🚀 Идея проекта

### Концепция
AstroBit исследует гипотезу о возможном влиянии космических явлений на финансовые рынки, предоставляя уникальную платформу для:
- **Корреляционного анализа** между астрономическими событиями и криптовалютными движениями
- **Исследования цикличности** рынков в контексте космических циклов
- **Визуализации данных** для выявления скрытых паттернов
- **Научного подхода** к астрологическому анализу финансов

### Уникальность
- 🌌 **Первая платформа** такого типа, объединяющая астрономию и криптотрейдинг
- 🔬 **Научная основа** с использованием профессиональных астрономических библиотек
- 📊 **Real-time интеграция** с криптовалютными биржами
- 🎯 **Исследовательский фокус** на выявлении корреляций, а не предсказаниях

## 🎯 Целевая аудитория

- **Криптотрейдеры** — анализ рыночных движений в контексте космических событий
- **Исследователи** — изучение потенциальных влияний космоса на финансовые рынки
- **Астрологи** — применение астрологических принципов к финансовому анализу

- **Академические круги** — исследование междисциплинарных связей

## 🏗 Архитектурные паттерны

### Clean Architecture
Проект построен на принципах Clean Architecture с четким разделением на слои:

```
┌─────────────────────────────────────────────────────────────┐
│                    Presentation Layer                       │
│              (React Components, UI, Hooks)                 │
├─────────────────────────────────────────────────────────────┤
│                    Application Layer                        │
│              (Use Cases, Application Services)             │
├─────────────────────────────────────────────────────────────┤
│                     Domain Layer                            │
│           (Entities, Value Objects, Business Logic)        │
├─────────────────────────────────────────────────────────────┤
│                  Infrastructure Layer                       │
│           (External APIs, Repositories, Services)          │
└─────────────────────────────────────────────────────────────┘
```

### Domain-Driven Design (DDD)
- **Entities** — `AstronomicalEvent`, `CryptoData`, `Chart`
- **Value Objects** — `EventSignificance`, `EventType`, `Symbol`, `Timeframe`
- **Repositories** — `IAstronomicalEventRepository`, `ICryptoDataRepository`
- **Services** — `IAstronomicalCalculationService`

### Паттерны проектирования
- **Repository Pattern** — абстракция доступа к данным
- **Use Case Pattern** — бизнес-операции и координация
- **Dependency Injection** — инверсия контроля и тестируемость
- **Observer Pattern** — real-time обновления данных

## 🚀 Быстрый старт

### Требования
- **Node.js** >= 14.0.0
- **npm** или **pnpm** (рекомендуется)

### Установка
```bash
# Клонирование репозитория
git clone https://github.com/scoutrul/astrobit.git
cd astrobit

# Установка зависимостей
npm install
# или
pnpm install

# Запуск dev-сервера
npm run dev
# или
pnpm dev
```

### Доступные команды
```bash
npm run dev          # Запуск dev-сервера
npm run build        # Сборка для продакшена
npm run lint         # Проверка качества кода
npm run preview      # Предварительный просмотр сборки
```
